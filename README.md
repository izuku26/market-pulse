# Global Market Pulse

Un dashboard de Business Intelligence simple pour comparer la performance d'actifs financiers (Actions, Crypto, Or, Forex) depuis 2012.

## Fonctionnalités
- **Pipeline ETL :** Script Python automatisé pour la récupération et le nettoyage des données Yahoo Finance.
- **Analyse BI :** Visualisation des rendements cumulés (Base 100), du rapport Risque/Rendement et des chutes maximales (Drawdown).
- **Saisonnalité :** Heatmap des performances mensuelles moyennes.
- **Interface :** Dashboard interactif réalisé avec Streamlit et Shadcn UI.

## Stack Technique
- **Langage :** Python 3.10+
- **Données :** Pandas, YFinance, PyArrow (Format Parquet)
- **Visualisation :** Plotly, Streamlit
- **Automatisation :** GitHub Actions (MAJ quotidienne des données)
